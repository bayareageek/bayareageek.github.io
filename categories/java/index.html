Java 虚拟机原理 (六) ZGC 垃圾收集器

2021-04-08 发布在 Java

ZGC 是 JVM 生态最新最先进的一款垃圾收集器，具有可扩展低延迟高吞吐量等特性，其设计目标包括：

  • 亚毫秒级停顿时间（ <= 10ms)；
  • 停顿时间长短与堆大小、存活对象数量、根节点数量无关；
  • 支持堆大小从8MB到16TB不等；
